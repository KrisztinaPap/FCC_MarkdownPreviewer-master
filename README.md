# FreeCodeCamp Markdown Previewer Project

A solution for FreeCodeCamp's Markdown Previewer front-end project.

## Features:

- Utilized HTML5, CSS3, JQuery, Bootstrap, JavaScript, and the Markdown library.

## Resources:
